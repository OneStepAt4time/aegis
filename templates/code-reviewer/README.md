# `code-reviewer`

- **Type:** agent
- **Scaffolds:** `.claude/agents/code-reviewer.md`
- **Command:** `ag init --from-template code-reviewer`

Use this starter when you want a repository-local review agent focused on
correctness, regressions, risky edge cases, and missing tests.

The generated file includes a customization checklist, a default review
workflow, and an output format you can tailor to your project.
